About the code

This helloworld.py is typical of the scripts we will wind up with.

    #!/usr/bin/env python3
    import urllib.request
    import json
    import getpass
    # additional modules get added here.
    
    # This is where we add our functions
    def hello_world():
        return 'Hello World!!!'
    
    # This is where we put our tests
    if __name__ == '__main__':
        hw = hello_world()
        if hw == 'Hello World!!!':
            print('Success!! -> ', hw)
        else:
            print('Ooops something went wrong!!!"')

You shold see “Success!! -> Hello World!!!” when you run the program.
